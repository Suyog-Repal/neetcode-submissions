# NeetCode 250 Solutions — Suyog Repal

This repository contains my solutions to problems from the **NeetCode 250 roadmap**, a curated list of important Data Structures and Algorithms problems designed for coding interview preparation.

## About NeetCode

[NeetCode](https://neetcode.io) is a well-known platform for coding interview preparation that organizes important problems by topic such as:

- Arrays & Hashing
- Two Pointers
- Sliding Window
- Stack
- Binary Search
- Linked Lists
- Trees
- Graphs
- Dynamic Programming
- Greedy
- and more.

The roadmap covers the most important patterns commonly asked in technical interviews.
